Steps to compiling project on Windows Terminal
	1. Make sure all 6 files are in the same directory
		a. Project2.java
		b. InformationDesk.java
		c. Announcer.java
		d. Agent.java
		e. Customer.java
		f. SemFunctions.java
	
	2. Run the following commands in order
		a. Java compile - javac Project2.java
		b. Java run - java Project2

	1. Connect using GlobalProtect
	2. Open an SSH client (I used XShell)
